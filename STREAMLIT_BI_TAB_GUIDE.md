## Data Integration

The Business Intelligence tab automatically loads:

1. **Latest Analysis Results** (`patent_analysis_results_*.json`)
   - All 167 patents with complete scoring
   - Technical profiles, financial models, strategic assessments
   
2. **Rankings CSV** (`patent_rankings_*.csv`)
   - Quick-reference table with all key metrics
   - Tier classifications and red flags

3. **Markdown Report** (`EXPIRED_PATENT_REPORT_*.md`)
   - Full business intelligence report
   - Downloadable for stakeholder sharing

If any file is missing, tab displays helpful message: "No Business Intelligence analysis available yet. Run `python run_expired_patent_analysis.py` to generate."
